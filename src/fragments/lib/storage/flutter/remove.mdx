## Remove a file

You can remove a single file using `Amplify.Storage.remove` with the `path`:

```dart
import 'package:amplify_flutter/amplify_flutter.dart';

Future<void> removeFile({
  required StoragePath path,
}) async {
  try {
    final result = await Amplify.Storage.remove(
      path: path,
    ).result;
    safePrint('Removed file: ${result.removedItem.path}');
  } on StorageException catch (e) {
    safePrint('Error deleting file: ${e.message}');
    rethrow;
  }
}
```

## Remove multiple files

You can remove multiple files using `Amplify.Storage.removeMany` with the `paths`:

```dart
import 'package:amplify_flutter/amplify_flutter.dart';

Future<void> removePrivateFiles({
  required List<StoragePath> paths,
}) async {
  try {
    final result = await Amplify.Storage.removeMany(
      paths: paths,
    ).result;
    safePrint('Removed files: ${result.removedItems}');
  } on StorageException catch (e) {
    safePrint('Error deleting files: ${e.message}');
    rethrow;
  }
}
```
